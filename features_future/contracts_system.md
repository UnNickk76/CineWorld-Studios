# Sistema Contratti - Game Design Document

## Panoramica
Sistema di contratti tra giocatori, studi e attori per creare accordi commerciali nel mondo di CineWorld.

## Funzionalita Previste
- Contratti a tempo con attori/registi (esclusiva, multi-film)
- Contratti di distribuzione tra studi
- Penali per rottura contratto
- Negoziazione con controproposta
- Contratti speciali per Star (legati al sistema Star Discovery)

## Impatto sull'Economia
- I contratti bloccano fondi come garanzia
- Le penali alimentano il mercato
- Gli attori sotto contratto esclusivo non sono disponibili per altri studi

## Fasi di Implementazione
1. Schema DB contratti
2. Endpoint backend CRUD + validazione
3. UI frontend per stipula e gestione
4. Integrazione con auto-tick per scadenze automatiche
